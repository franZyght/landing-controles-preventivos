Controles Preventivos

A partir del jueves 1ro. de agosto de 2019 tendremos dos mejoras importantes enel módulo Controles Preventivos. La primera funcionalidad se va a enfocar en Listas de Verificación (checklist)  y Caminatas de Seguridad (inspecciones) y la segunda, se trata de la generación de Hallazgos en las respuestas de los Checklist. Conoce más detalles a continuación.

Funcionalidad 1

Reportar Actividades Planificadas de Manera Grupal

A partir del jueves 1ro. de Agosto, podrán reportarse actividades planificadas de tipo Listas de Verificación (Checklists) y Caminatas de Seguridad (Inspecciones) de forma grupal.

De esta manera, si un usuario realiza la actividad con acompañantes que tienen el mismo tipo de actividad planificada para la misma fecha, se descontará la actividad tanto para el usuario que reporta, como para sus acompañantes.

¿Cómo se Reporta?

Paso 1 
 Al momento de reportar una Lista de Verificación o Caminata de Seguridad, se debe indicar si el reporte se realiza con acompañantes.

Paso 2 
Si el usuario selecciona la opción Sí, podrá escoger los acompañantes del usuario que reporta.

Funcionalidad 2

Generar Hallazgos a través de Checklist

A partir del jueves 1ro. de Agosto, las respuestas de las checklists podrán generar hallazgos independiente de la ponderación de la respuesta. De esta manera, se agrega el campo Hallazgo, para que el usuario indique si la respuesta de la checklist puede o no, generar hallazgo.
